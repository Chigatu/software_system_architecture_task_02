🔐 Аутентификация: Для защищенных эндпоинтов используйте кнопку "Authorize" и введите токен в формате: Bearer <your-token>

📝 Получить токен: Сначала зарегистрируйтесь через POST /api/auth/register, затем выполните POST /api/auth/login

💾 Хранилище: Данные хранятся в памяти, при перезапуске сервера все данные сбрасываются
